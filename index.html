Tutorial base: parsing data on a worker

Your goal in this session is to take some complex data, parse it, and fill in the table below to show how many Firefox users are running Windows.

The data is quite large, and so parsing it using a standard script will cause significant browser hangs. Instead, you are expected to parse it on a worker thread using a DOM Worker.

The data is available at http://analysis-output.telemetry.mozilla.org/stability-rollups/2015/20151128-active-weekly.csv.gz . Here is the top of that file:
